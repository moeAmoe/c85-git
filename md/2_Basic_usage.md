Basic usage
======
基本的な操作を説明する。

## git init
現在のディレクトリ以下のgit管理を開始する。

`.git` 以下がリポジトリの実体である。

## git status
現在のワーキングツリー／ステージングの状態を確認する。

## git add
対象ファイルをステージングに移す。

## git rm
対象ファイルを削除する。

## git mv
対象ファイルを移動／名前変更する。

## git commit
ステージングにあるファイルをリポジトリに適用する。

## git branch
ブランチコマンド

## git checkout
ブランチを変更する。
